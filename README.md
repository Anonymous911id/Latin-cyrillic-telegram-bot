# Uzbek Transliterate Telegram Bot

This is a **Telegram bot** that allows users to transliterate Uzbek text between **Latin and Cyrillic scripts**.

The bot uses the `transliterate` function from the https://github.com/kodchi/uzbek-transliterator repository as the core transliteration engine.

## Features

- Transliterate Uzbek text from **Cyrillic to Latin**
- Transliterate Uzbek text from **Latin to Cyrillic**
